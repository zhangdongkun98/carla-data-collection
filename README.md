# carla-data-collection

### installation

- install [carla-utils](https://github.com/zhangdongkun98/carla-utils.git)

```bash
pip install open3d==0.8.0.0
```

### collect data

- collect

```bash
python run.py --save
```

- visualize data

```bash
python run.py --render
```



### load collected data

```bash
python load.py
```
